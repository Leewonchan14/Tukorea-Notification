## 설치 및 실행 방법

### 요구 사항

- Docker
- Docker Compose

### 설치

1. 저장소를 클론합니다.

   ```sh
   git clone https://github.com/your-repo.git
   cd your-repo
   ```

2. Docker 컨테이너를 빌드하고 실행합니다.
   ```sh
   docker-compose up --build
   ```
